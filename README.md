# SQL Online Store Sales Analysis
## Project Overview
SQL Online Store Sales Analysis project aims at showcasing the capabilities of using SQL for database development and analysis of the sales data of an online store.
## Database Structure
The database for this project has been named **OnlineStore**, and the tables within the database are as follows:
*   **Customers**: This table holds the information of the customers.
*   **Products**: This table holds the information of the products and their prices.
*   **Orders**: This table holds the information of the orders made in the store.
*   **Order_Details**: This table holds the information of the order details for each order made in the store.
